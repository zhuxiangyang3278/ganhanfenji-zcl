# 基于Restormer的三分支干旱分级网络架构文档

## 📊 总体架构概览

```
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│    RGB输入      │  │  热红外输入     │  │  多光谱输入     │
│   (3通道)       │  │   (3通道)       │  │   (8通道)       │
└─────────────────┘  └─────────────────┘  └─────────────────┘
         │                     │                     │
┌─────────────────┐  ┌─────────────────┐  ┌─────────────────┐
│ Restormer编码器  │  │ Restormer编码器  │  │ Restormer编码器  │
│   (RGB分支)     │  │   (热红外分支)   │  │   (多光谱分支)   │
└─────────────────┘  └─────────────────┘  └─────────────────┘
         │                     │                     │
┌─────────────────────────────────────────────────────────┐
│             特征拼接与融合层                            │
│   (三个分支的base特征在通道维度拼接)                    │
└─────────────────────────────────────────────────────────┘
                         │
┌─────────────────────────────────────────────────────────┐
│             全局特征提取                                │
│   (全局平均池化 GAP)                                   │
└─────────────────────────────────────────────────────────┘
                         │
┌─────────────────────────────────────────────────────────┐
│             分类头                                     │
│   (全连接层: 192 → 256 → 5)                           │
└─────────────────────────────────────────────────────────┘
                         │
┌─────────────────────────────────────────────────────────┐
│             输出: 5类干旱等级                           │
│   (Level 0-4)                                          │
└─────────────────────────────────────────────────────────┘
```

## 🎯 核心设计理念

### 1. 多模态信息互补
- **RGB模态**: 提供地表纹理、颜色等视觉特征
- **热红外模态**: 提供地表温度信息，反映水分状况
- **多光谱模态**: 提供植被指数，反映植被健康状况

### 2. Restormer架构优势
- **计算效率**: 深度可分离卷积减少计算复杂度
- **特征提取能力**: 同时捕获全局和局部信息
- **图像适应性**: 专门为图像处理优化的Transformer架构

### 3. 三分支独立编码
- 每个模态使用独立的编码器，保留模态特性
- 避免模态间信息干扰
- 便于后续的特征级融合

## 🔍 详细架构解析

### 1. 输入层设计

| 输入分支 | 通道数 | 数据内容 | 作用 |
|---------|--------|----------|------|
| RGB分支 | 3通道 | 可见光图像 | 地表纹理、颜色特征 |
| 热红外分支 | 3通道 | 热红外图像 | 地表温度、水分状况 |
| 多光谱分支 | 8通道 | 多光谱+植被指数 | 植被健康状况 |

### 2. Restormer编码器架构

```python
class Restormer_Encoder(nn.Module):
    def forward(self, inp_img):
        inp_enc = self.patch_embed(inp_img)          # 重叠补丁嵌入
        out_enc = self.encoder_blocks(inp_enc)       # Transformer块
        base_feature = self.base_layer(out_enc)      # 基础特征提取
        detail_feature = self.detail_layer(out_enc)  # 细节特征提取
        return base_feature, detail_feature, out_enc
```

**核心组件:**
- **MDTA注意力**: 多深度卷积头转置自注意力机制
- **GDFN前馈**: 门控深度卷积前馈网络
- **LayerNorm**: 带偏置的层归一化

### 3. 特征层次分离

每个Restormer编码器输出两个层次的特征：

#### Base Feature (基础特征)
- **作用**: 提取全局低频信息
- **特点**: 关注整体模式、大范围特征
- **应用**: 用于后续的特征融合和分类

#### Detail Feature (细节特征)  
- **作用**: 提取局部高频信息
- **特点**: 关注细节纹理、边缘信息
- **应用**: 可用于后续的细节分析

### 4. 特征融合策略

```python
# 三个分支的base特征拼接
fused_features = torch.cat([base1, base2, base3], dim=1)  # (B, 192, H, W)

# 全局平均池化  
global_features = F.adaptive_avg_pool2d(fused_features, 1)  # (B, 192, 1, 1)

# 分类头
output = self.classifier(global_features.view(global_features.size(0), -1))
```

**融合特点:**
- **通道维度拼接**: 保留各模态的独立特征表达
- **全局平均池化**: 将空间特征转换为全局描述子
- **特征维度**: 3分支 × 64维度 = 192维全局特征

### 5. 分类头设计

```
192维特征 → 256维隐藏层 → 5类输出
    ↓           ↓           ↓
全局特征 → 特征增强 → 干旱等级分类
```

| 分类头 | 192→256→5 | 两层全连接 |

## 🎯 技术亮点与创新

### 1. 多模态Restormer架构
- **创新点**: 首次将Restormer应用于多模态干旱分级
- **优势**: 结合了Restormer的计算效率和Transformer的特征提取能力

### 2. 三分支独立编码设计
- **创新点**: 每个模态使用独立编码器
- **优势**: 避免模态间信息干扰，保留各模态特性

### 3. 特征层次分离
- **创新点**: base/detail特征分离提取
- **优势**: 同时捕获全局模式和局部细节

### 4. 自适应复杂度调整
- **创新点**: 根据任务复杂度自动调整模型配置
- **优势**: 平衡模型表达能力和计算效率
